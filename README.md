📊 HR Analytics Dashboard
🚀 Project Overview

The HR Analytics Dashboard is a data-driven project designed to analyze employee data and provide actionable insights for better workforce management.
This dashboard helps HR teams understand trends related to employee attrition, performance, and overall workforce distribution.

🎯 Objectives
Analyze employee attrition and identify key factors
Monitor workforce distribution across departments
Track employee performance and satisfaction levels
Support data-driven HR decision making
🛠️ Tools & Technologies Used
Python (Data Processing)
Pandas (Data Analysis)
Power BI / Streamlit (Dashboard Visualization)
SQL (Data Extraction)
Excel (Data Source & Cleaning)
📂 Dataset

The dataset contains HR-related information such as:

Employee ID
Age, Gender, Department
Salary, Job Role
Attrition Status
Performance Ratings
Years at Company
📈 Key Insights
Identified departments with high attrition rates
Analyzed impact of salary and experience on attrition
Observed trends in employee satisfaction levels
Highlighted key factors influencing employee retention
📊 Dashboard Features
Interactive visualizations
Attrition analysis by department and role
Employee demographics overview
Performance tracking charts
Filter options for dynamic analysis
